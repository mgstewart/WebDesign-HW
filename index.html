Summary

The purpose of this project is to assess whether there is a relation between proximity to the equator and the max temperature recorded on a given day in random cities around the world. To perform this, a database of cities with latitude and longitude was acquired (see the Data page). We loaded the cities into a pandas dataframe, removed exact duplicates of lat/long coordinates, and selected 1000 random cities for querying.

Using the OpenWeatherMap API and city names, we added temperature, wind speed, cloud coverage, and humidity into the dataframe of random cities. Using Matplotlib, we created visualizations of the trends of these data against latitude of the cities. This resulted in five distinct visualizations, which can be accessed on this website (to the right of your screen or below, for mobile users!)
